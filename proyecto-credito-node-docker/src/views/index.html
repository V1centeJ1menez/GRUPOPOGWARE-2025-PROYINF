Grupo Pogware

Plataforma Web para Solicitud de Créditos Bancarios. El objetivo del proyecto es ofrecer a los clientes una solución en línea para gestionar solicitudes de crédito, reduciendo tiempos de espera y mejorando la transparencia en el seguimiento del estado de cada solicitud.
